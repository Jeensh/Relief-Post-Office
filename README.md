## 앱 설치하기
안드로이드 폰에 아래 apk파일 설치 후 실행  
https://github.com/Relief-Post-Office/Relief-Post-Office/blob/main/app-release.apk?raw=true 

## 프로젝트 목적
**사회적 취약 계층**과 복지 종사자 및 지역 사회의 일원들을 연결하여  
취약 계층에게 **정서적인 지원**과 **지속적인 관리**를 제공할 수 있는 새로운 솔루션을 개발하기 위함

- Android Studio Chipmunk
- Firebase
  - Realtime Database
  - Storage
  - Authentication
  - Cloud Messaging
  - In-App Messaging
  - Hosting

- targetSdkVersion : 32
